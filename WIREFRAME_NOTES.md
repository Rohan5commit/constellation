# Wireframe Notes

## Low-Fi Structure Rationale

The prototype is designed as a sequence of calm, focused screens instead of a single dense dashboard.

## Screen Logic

### Landing
- immediate emotional framing
- strong celestial metaphor
- short explanation of fragmented memory
- clear CTA into prototype

### Onboarding
- three-step explanation: capture, cluster, rediscover
- lightweight enough for judges to process fast

### Demo Feed
- one strong resurfaced item above the fold
- supporting fragments below
- visible path into constellation view

### Constellation View
- orbit-based grouping to reinforce the metaphor
- grouping rationale placed adjacent to the map
- fragment nodes large enough to scan quickly

### Fragment Detail
- clear answer to “why is this back?”
- context threads: person, project, place, emotion, timing
- one primary next step

### Reconnect Flow
- gentle language
- no manipulative urgency
- message suggestions short enough to feel usable

### Reflection
- emotional payoff screen
- designed to close the demo on meaning, not mechanics

## Visual Density Rules

- one main narrative action per screen
- no tiny text
- no overpopulated graphs
- decorative stars only where clarity is preserved
