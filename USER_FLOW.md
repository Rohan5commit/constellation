# User Flow

## Primary Demo Journey

1. Land on the story-led homepage
2. Understand that the product is about recovering lost fragments of life and work
3. Enter onboarding
4. Move into seeded demo mode
5. See resurfaced fragments in the rediscovery feed
6. Open a constellation to understand grouping
7. Open a fragment detail page
8. Read “why this resurfaced” context
9. Choose an action: act, reconnect, archive, or let go
10. End on a reflection or reconnection payoff

## Capture Flow

1. Tap “Capture a fragment”
2. Choose fragment type
3. Add lightweight content
4. Receive calm confirmation
5. See the fragment suggested into a constellation

## Constellation Flow

1. Open cluster view
2. Scan grouped fragments in orbit
3. Read grouping rationale
4. Select one fragment for detail

## Reconnect Flow

1. Open a people-related fragment
2. See the relationship context
3. Review low-pressure message suggestions
4. Decide whether to reach out now, snooze, or let it rest

## Weekly Reflection Flow

1. Open reflection screen
2. Read the pattern the system noticed this week
3. Review “found again” moments
4. Carry one thought forward
